# react-virtualized_InfiniteLoader_WindowScroller

## Overview
POC demonstrating dynamic content loading based on window scroll position using react-virtualized. This project explores efficient rendering patterns for large datasets through virtual scrolling and infinite loading techniques.

## Key Implementations
- **InfiniteLoader**: Loads items dynamically as user scrolls
- **WindowScroller**: Integrates with native browser scroll
- **Masonry Layout**: Pinterest-style grid with dynamic heights
- **AutoSizer**: Responsive component sizing

## Usage
```bash
pnpm install
pnpm start
```

Built with React 16.14 and react-virtualized 9.22.6
